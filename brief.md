## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

### Общий портрет
Люди 30–55 лет, которые **уже не могут жить по-старому**: работают, вкладываются, но бизнес не даёт прибыли, прежняя реальность закончилась, а как действовать в новой — непонятно. Ищут не просто отдых, а внутреннюю цельность.

### Три сегмента

**I. «Искатель опоры в тонком»**
- Медитации, энергии, ретриты — его мир. Многое понял про себя.
- Боль: деньги утекают, бизнес не растёт, разрыв между «высоким» и «низким».
- Что даёт Сатори: соединяет поиск и результат, заземляет практики.

**II. «Хозяин твёрдой ниши»**
- Работает в традиционном бизнесе, много трудится, держит удар.
- Боль: старые методы не работают, новые — непонятны, прежние усилия не дают прибыли.
- Что даёт Сатори: новая опора без эзотерики, ясность направления.
- ⚠️ Именно на этот сегмент рассчитан скептицизм — ключевой объект работы лендинга.

**III. «Интегратор — в поиске целостности»**
- Есть и бизнес, и понимание энергий. Всё вроде неплохо.
- Боль: тонкое внутреннее сомнение «А точно ли я на правильном пути?».
- Что даёт Сатори: укрепляет цельность, даёт внутренний компас.

### Психографика (из оценки Эксперта 3)
Скептичный руководитель 35–50 лет за первые 5 секунд смотрит на:
1. Фото (реальное или сток?)
2. Заголовок (нет ли «трансформации / пробуждения»?)
3. Дату и место (Майские? Карелия? Интересно)
4. Цену (если спрятана — подозрение)
5. Ведущего (ищет «не сектант»)

**Главное возражение:** «Я не уверен, что это мне поможет. И не уверен, что там адекватные люди.»

- **Smooth scroll:** `scroll-behavior: smooth` + учёт высоты header (`offset = headerHeight + 16px`)

### Кастомный курсор

```
.cursor         — точка 8px, цвет акцента
.cursor-follower — окружность 32px, полупрозрачная граница
```

- Работает только на десктопе (> 768px)
- При hover на кнопках/карточках: follower увеличивается до 48px
- JS: follower движется с задержкой (lerp `0.12`)

---

### Scroll Reveal

- Все секции имеют класс `.reveal`
- IntersectionObserver: порог 12%, rootMargin `0px 0px -40px 0px`
- При пересечении: добавляется `.visible`, убирается из observer
- CSS: `opacity: 0; transform: translateY(28px)` → `opacity: 1; transform: none`
- Transition: `0.7s ease`

---

### Parallax

- `.hero-bg`: `translateY(scrollY * 0.3)` — работает до 1.5 * viewport height
- `.hero-water`: `translateY(scrollY * 0.15)` — более медленный слой
- Пассивный listener: `{ passive: true }`

---

### FAQ-аккордеон

- При клике на `.faq-question`: устанавливается `maxHeight = scrollHeight`
- Предыдущий открытый элемент закрывается
- Иконка `+` → `×` через `rotate(45deg)`
- Transition: `max-height 0.4s ease`

## 7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### Технологический стек
- **HTML5** — семантическая разметка
- **CSS3** — Custom Properties, Grid, Flexbox, анимации

- **Без:** jQuery, React, WebGL, Three.js, тяжёлых библиотек

### Адаптивность (mobile-first)

| Breakpoint | Поведение |
|---|---|
| `> 1024px` | Полный десктоп, все колонки |
| `≤ 1024px` | Программа 2×2, pricing 1 колонка |
| `≤ 768px` | 1 колонка везде, мобильное меню |
| `≤ 480px` | Мелкие корректировки |

> ⚠️ **Приоритет:** mobile-first. Верстать начиная с мобильного, расширять для десктопа.

### Производительность
- Изображения: WebP-формат, lazy loading (`loading="lazy"`)
- Шрифты: `display=swap` в Google Fonts URL
- Видео (если есть): `autoplay muted loop playsinline`, без звука
- Без WebGL и Three.js — скорость загрузки важнее
- Цель: загрузка < 3 секунд (каждая секунда сверх = –7% конверсии)

### Доступность

- `alt`-тексты для всех изображений
- `aria-label` для кнопки бургер-меню

## 8. ЗАПРЕЩЁННЫЕ АНТИПАТТЕРНЫ

*Синтез замечаний трёх экспертов. Нарушение любого из пунктов снижает конверсию.*

### Контент
| Нельзя | Почему |
|---|---|
| Слова «трансформация», «пробуждение», «путешествие к себе» без конкретики | Клише, вызывает недоверие у скептика |
| Скрытые или отсутствующие цены | Воспринимается как манипуляция |
| Стоковые фото (женщины в белом у воды) | Убивает аутентичность — критично для ниши |
| Анонимные отзывы без фото («Анна, Москва») | Не работают |
| Слово «ченнелинг», «высшие силы» без пояснения | Ассоциация с сектой |

### UX/поведение
| Нельзя | Почему |
|---|---|
| Автовоспроизведение видео со звуком | Немедленный выход, особенно на мобиле |
| Попап в первые 3 секунды | Воспринимается как агрессия |
| Попап при выходе («Уходите? Подождите!») | Отталкивает думающую аудиторию |
| Фиктивные таймеры обратного отсчёта | Манипуляция, разрушает доверие |
| Горизонтальный скролл на мобиле | Один из главных UX-грехов |
| Форма только в самом низу страницы | Потеря готового к конверсии пользователя |
| Текст без состояния загрузки после отправки | Пользователь не знает, сработало ли → уходит |
| Низкий контраст текста на светлом фоне | Нечитаемо → раздражение → уход |

### Технические
| Нельзя | Почему |
|---|---|
| WebGL / Three.js | Медленная загрузка, нет в требованиях |
| Блог / дополнительные страницы | Только лендинг |
| Онлайн-оплата | Не в рамках этого проекта |
| Чат-бот | Заменяется Telegram |
| Мультиязычность | Только русский |
